## 핵심 개념 (Pick 3)

1. **배열 vs 객체의 구조와 쓰임**
   - 배열: 순서(인덱스)기반 자료 저장, 반복/정렬에 강점
   - 객체: key-value(이름:값) 쌍 저장, 자료 이름 부여, 중첩/확장 유연
   - 배열/객체 조합과 중첩 자료구조 완벽히 이해하면 복잡한 데이터도 구조화 가능
2. **데이터 바인딩과 클라이언트 사이드 렌더링(CSR)**
   - 서버에서 html을 모두 완성해 보내는 SSR, 데이터만 받아와 JS가 직접 DOM생성하는 CSR
   - 대부분의 동적 웹서비스에서 CSR 기반 데이터 바인딩(템플릿에 JS 데이터 입력)이 주력
   - createElement, innerHTML, appendChild 등 실제 DOM 조작 패턴 실습
3. **중첩된 자료에서 값 꺼내는 방법/패턴 훈련**
   - 객체 속 배열, 배열 속 객체 등에서 값을 꺼낼 때 dot+[index] 활용
   - 실제로 data가 복잡해도 “찍고->변환->대입” 반복하며 익숙해지는 것이 관건

## 코드/실험 & 깨달은 점

### 1. **상품 데이터 객체 배열 → 카드 HTML 자동 생성**

```js
var products = [
  { id: 0, price: 70000, title: 'Blossom Dress' },
  { id: 1, price: 50000, title: 'Springfield Shirt' },
  { id: 2, price: 60000, title: 'Black Monastery' }
];

let cardlist = document.querySelector('.card-group');

for (let i = 0; i < products.length; i++) {
  let item = document.createElement('div');
  item.innerHTML = `
    <div class="card">
      <img src="https://placehold.co/600">
      <div class="card-body">
        <h5>${products[i].title}</h5>
        <p>가격 : ${products[i].price}</p>
        <button class="btn btn-danger">주문하기</button>
      </div>
    </div>
  `;
  cardlist.appendChild(item.firstElementChild);
}
```
- 객체(data) 기반 for문으로 동적 html 생성
- innerHTML로 템플릿 만들고, createElement/appendChild로 DOM에 바로 붙이기

### 2. **객체 내부의 배열 데이터 바인딩**

```js
let car2 = {
  name: '소나타',
  price: [50000, 3000, 4000],
};

document.querySelector('.car-price').innerHTML = `가격: ${car2.price[0]}`;
```
- 객체/배열 혼합자료에서 값 빼내기(Dot+[index])에 완전히 익숙해지기
- 배열·객체 중첩도 하나씩 찍으며 구조 파악하면 쉽게 처리 가능
