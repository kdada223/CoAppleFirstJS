## 핵심 개념 (Pick 3)

1. **동적 select 옵션 생성(실무형 데이터 바인딩)**
   - 상품 카테고리(셔츠/바지 등) 선택 시 forEach로 배열 데이터 기반 option을 동적으로 삽입
   - 기존 수동 newselect/문자열 방식 없이, 데이터만 교체하면 알아서 HTML 변경됨
2. **forEach와 for in 반복문의 차이 완전 체득**
   - forEach: 배열 순회, (item, idx, arr) 콜백/인덱스 활용, for문보다 안전하고 익숙해지기 좋음
   - for in: 객체의 key 순회 전용, obj[key]로 value 추출. 배열에 쓸 때 주의(예상치 못한 키 등장 가능)
3. **화살표 함수와 function의 this 차이**
   - function은 “호출 위치”에 따라 this 바인딩
   - 화살표 함수는 “선언 당시 스코프”의 this를 항상 그대로 사용  
   - 이벤트·콜백 함수 내부에서 this의 차이 직접 실험/확인함

## 코드/실험 & 깨달은 점

### 1. **동적 옵션 생성 및 반복문 패턴**

```js
let pants = [28, 30, 32, 34];
let shirts = [90, 95, 100];
document.querySelector('.form-select').addEventListener('change', function (e) {
	let secondSelect = document.querySelectorAll('.form-select')[1];
	let target = e.target.value;
	if (secondSelect) secondSelect.innerHTML = '';
	if (target === '셔츠') {
		shirts.forEach(item => {
			secondSelect.insertAdjacentHTML('beforeend', `<option>${item}</option>`);
		});
	} else if (target === '바지') {
		pants.forEach(item => {
			secondSelect.insertAdjacentHTML('beforeend', `<option>${item}</option>`);
		});
	}
});
```
- forEach로 옵션별 반복, insertAdjacentHTML로 간단히 동적 생성  
- 어떤 카테고리든 배열만 바꾸면 확장 가능

### 2. **객체 순회(for in), 키/값 추출 실습**

```js
let obj = { name: 'kim', age: 20 };
for (let key in obj) {
	console.log(key);      // name, age
	console.log(obj[key]); // kim, 20
}
```
- key와 value를 모두 출력, 객체 구조에 치명적으로 익숙해짐

### 3. **this 바인딩 차이(arrow vs function) 실험**

```js
let testObj = {
	name: "Lee",
	showThis_Arrow: () => {
		console.log('arrow this:', this); // window(최상위 or 상위스코프)
	},
	showThis_Function: function() {
		console.log('function this:', this); // 호출되는 객체(testObj)
	}
};
testObj.showThis_Arrow();
testObj.showThis_Function();
```
- 직접 차이 실험하니 네이티브 function의 동적 this, arrow의 렉시컬 this 완전히 이해
