## 핵심 개념 (Pick 3)

1. **DOM이란?**
   - 브라우저가 HTML을 읽을 때 각 엘리먼트와 속성을 JS object처럼 구조화한 "트리 형태의 데이터"이며, document 변수에 담겨 있어 자바스크립트로 조작 가능하다.
   - HTML을 JS에서 효율적으로 제어할 수 있게 한 중간 매개체이자, 점(dot) 문법으로 내부 속성에 접근/수정할 수 있게 한다.
2. **DOM 생성 시점과 JS 코드 실행 순서**
   - 브라우저는 HTML을 위에서 아래로 읽으며 DOM을 생성하기 때문에, 아직 읽지 않은 태그의 id/class에 JS가 먼저 접근하면 에러가 난다.
   - DOMContentLoaded, $(document).ready 등 이벤트리스너로 전체 DOM 생성 후 JS 코드를 실행하면 문제를 예방할 수 있다.
3. **고차배열 메서드, 함수화 실습**
   - filter/map/sort의 쓰임새(데이터 추출/가공/정렬)와 forEach, 반복문 함수화, 비동기 데이터처리(fetch) 흐름까지 종합 실습

## 코드/실험 & 깨달은 점

### 1. **DOM 생성 시점 실험과 오류**

```js
// 잘못된 예
<script>
  document.getElementById('test').innerHTML = '안녕'
</script>
<p id="test">임시글자</p>
// 오류 발생: JS가 p태그의 DOM 생성 전 실행됨!

// 올바른 예
<script>
  document.addEventListener('DOMContentLoaded', function () {
    document.getElementById('test').innerHTML = '안녕';
  });
</script>
<p id="test">임시글자</p>
```

- DOMContentLoaded 이벤트 사용 시, 위아래 코드 위치 상관없이 안전!

### 2. **filter/map/sort/카드동적생성/페이징 코드 반복 주석**

```js
let arr = [7, 1, 5, 2, 3, 40, 50];
arr.sort((a, b) => a - b); // 오름차순
arr.sort((a, b) => b - a); // 내림차순
let small = arr.filter((a) => a < 4); // 조건 필터
let times4 = arr.map((a) => a * 4); // 가공 map

var products = [
	{ id: 0, price: 70000, title: 'Blossom Dress' },
	{ id: 1, price: 50000, title: 'Springfield Shirt' },
	{ id: 2, price: 60000, title: 'Black Monastery' },
];

let cardlist = document.querySelector('.row');
function 포이치(runArr) {
	runArr.forEach((item) => {
		let secCardList = `<div class="col-sm-4">
      <img src="https://placehold.co/600" class="w-100">
      <h5>${item.title}</h5>
      <p>가격 : ${item.price}</p>
    </div>`;
		cardlist.insertAdjacentHTML('beforeend', secCardList);
	});
}
포이치(products);
// 기타 가격정렬/더보기(fetch) 버튼 등은 동일
```

### 3. **fetch POST 실습**

```js
fetch('https://codingapple1.github.io/hello.txt', {
	method: 'POST',
	headers: { 'Content-Type': 'application/json' },
	body: JSON.stringify({ name: '홍길동' }),
})
	.then((response) => response.json())
	.then((data) => {
		console.log(data);
	})
	.catch((error) => {
		console.log(error);
	});
```

- then-chain과 error catch 반복으로 ajax 흐름 체득!
